üìè Section 3: Understanding Story Points

Story points are used to estimate task complexity, not time. Use Fibonacci scale:

Story Points Meaning
1 Very simple, small bug fix.
2 Minor task, quick to complete.
3 Moderate task, some complexity.
5 Challenging but manageable.
8 Large task, needs discussion.
13+ Too big, must be split into smaller tasks.

üì° Section 4: Tracking Task Duration & WIP Limits

Enable WIP Limits in Jira:

  1. Go to Board Settings ‚Üí Columns.
  2. Set max limits (e.g., In Progress: 3, Code Review: 5).
  3. If exceeded, Jira highlights in red.

üìä Section 5: Reports & Metrics for Tracking

  • Control Chart: Tracks Cycle Time & Lead Time.
  • Cumulative Flow Diagram (CFD): Identifies bottlenecks.
  • Automation: Set alerts for tasks stuck for >5 days.

Use Jira automation to warn when tasks are taking too long.


Trigger: "Issue Transitioned to 'In Progress'"
Condition: JQL ‚Üí status = "In Progress" AND updated <= -5d
